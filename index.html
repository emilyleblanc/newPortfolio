My Work

Portfolio

  • Knit + Pearl

    Knit + Pearl is a mock-up yarn store. I built the inventroy using a firebase database and built the app with React. You can search the inventory by fiber, add products to your cart and receive a running total of your purchases. I'm looking forward to using this site to sell my own knitting creations one day soon.

  • Under the Sea

    This project is a multi-page site including a homepage, blog and contact form. It follows the style guide and PSD layout. In addition it is responsive and accessble so anyone can enjoy it on any device. In collaboration with Andrew Campbell.

  • Sun Run

    We successful built this app to the specifications of our client brief. We used React and REST-ful API so that the user can search for the Sunrise and Sunset given their time zone. Once you choose a date and timezone you can choose the length of your run and the app will let you know what time you should leave your house in order get home before dark. In collaboration with Rosie Foote, Jordan Chan and Tobi Okunuga

  • Raise the Bar

    You can try this app to search breweries across the US by state. It uses a REST-ful API call to retrieve data and display your search results. There is also a fun feature where each time you re-load the page a new beer pun will appear. In collaboration with Rebecca MacDonald.
